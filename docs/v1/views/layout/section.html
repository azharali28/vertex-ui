Section

Sections are layout primitives used to control content width, alignment, and horizontal spacing. They ensure consistency across sections and help maintain readable layouts on all screen sizes.

Basic

Use .vx-section.

  • Centers content using margin-inline: auto
  • Applied max-width
  • Adds horizontal padding
