Table

Tables display structured data in rows and columns. They are ideal for presenting large datasets with clarity and alignment.

Example

# First Last Username
1 Azhar Ali @aali
2 John Doe @jdoe

Variants

Classes can be used to color tables, table rows and individual cells using .table-* classes.

Bordered Table

If you have a table that is too wide, you can add a container element with overflow-x:auto around the table, and it will display a horizontal scroll bar when needed.
