Spacing

Spacing utilities control margin and padding to create consistent layouts and visual rhythm across your UI.

Spacing Tokens

Value CSS Token Size
xs --vx-space-xs 4px
sm --vx-space-sm 8px
md --vx-space-md 16px
lg --vx-space-lg 24px
xl --vx-space-xl 32px
2xl --vx-space-2xl 48px
3xl --vx-space-3xl 64px
